# Loremaster
A SillyTavern extension which allows you to dynamically and asynchronously generate lore entries to a chat lorebook. Intended for on-the-fly lore memory creation. Run into a favorite NPC or a cool location you want to save down? This extension will handle that.

Designed on the principle that your primary lorebook, as the ground truth between sessions, remains untouched.

Once you've set a connection profile, you can simply highlight the character, lore tidbit, or concept that you want the extension to generate a lorebook entry for, and click one of the buttons to generate. In the settings menu, you can even add your own prompts for lorebook generation.
